uv package manager is highly recommended in case you want to try this project
(ive also freezed the requirements.txt in case you re pip fan)

uv initialization guide : 
1) open the /lab8 directory in your shell
2) run     *uv sync*
3) w8 for virtual environment to download necessary packages
4) enable the virtual environment
5) open the lab8/shopBack directory in your shell
6) run the dj project with    *python manage.py runserver*
